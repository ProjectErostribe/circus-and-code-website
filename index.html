#CircusAndCode

A ♥ProjectErostribe♥ Thesis

Created by: PAIDAMOYO-JANET "My.L.E.S" AZEHKO

PROJECT UPDATE: #CircusAndCode IS AN ATTEMPT TO SPARK TECHNOLOGICAL INNOVATION THROUGH THE EXPLORATION OF CIRCUS ARTS

#CircusAndCode joins the Alpha Centauri App!

Join the fun at: https://twistedplanet.disciplemedia.com



THE PREQUEL:

Review JavaScript basics using Alchemy University (www.university.alchemy.com), and UX/UI via BRIDGEGOOD/CourseEra's Google's Certification.

ACT 1:

Conduct user research to figure out #CircusAndCode's target audience in order to redesign the #CircusAndCode website, game, and begin work on the #CircusAndCode capsule collection.

ACT 2:

Research and build an Airtable document that captures daily workout information and coding journey. This information will be used to build the #CircusAndCode online movebook.

ACT 3:

Learn the Who, What, When, Where and Why of Git.

ACT 4 :

Learn the art of Documentation using GitBook, GitHub CoPilot, and Docusaurus. Find out more about the #CircusAndCode capsule collection.

ACT 5 :

Build a DDR inspired game using JavaScript and Vue.js, #CircusAndCode music and a vintage DDR machine. Users can play on the web and on your iOS or Android device. Begin brainstorming on how to raise money for vintage DDR machine. (Sponsored by EveryLittleWar)

ACT 6 :

Build a backend for our game using JavaScript, Python and Ruby. Add to the music using Ruby Pi (Sponsored by Project Erostribe).

ACT 7 :

Design characters and build the online world of AlphaCentauri. (Inspired by Disciple Media).

ACT 8 :

Mint an NFT using Solana, Solidity and Motoko. Use your GitHub participation to win NFTs, swag, and more.

ACT 9 :

Design a sensory experience and learn more about clean eating as we look into how tech can be used to inspire healthy eating. (Inspired by Methodology).

ACT 10 :

Discover the effects of COVID-19 on the circus industry using Machine Learning and build a VR experience using Unity and C#.

ACT 11 :

Robotics and Wearables. Join My.L.E.S as we research how to make the jump from code to wearables.

ACT 12 :

Bullet Stopping Drone (Astral AR - Leah la Salla).

ACT 13 :

Rigging and Construction Tech

ACT 14 :

Flying Motorcycle and Drones

ACT 15 :

Build your act and build technical skills at the #CircusAndCode retreat.


ENCORE:

Compete for the chance to be part of #CircusAndCode LIVE! In Oakland, CA!
